# Git-document


## Initialization
### init:
```shell
$ git init
```


## Staging
### add:
```shell
$ git add <file-name>
$ git add -A
```
### reset:
```shell
$ git reset <file-name>
```

## Commit
### commit
```shell
$ git commit -m "<commit description>"
```

## Diff
### dif:
```shell
$ git diff HEAD
$ git diff --staged
```

## Movement
### checkout
```shell
$ git checkout <file-name>
$ git checkout -- <file-name>
```
